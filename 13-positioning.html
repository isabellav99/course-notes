Positioning

Positioning is one technique used to for moving elements on a page.

The position property takes one of the following values:

  • static
  • relative
  • fixed
  • absolute

static

Every element has a default value of static.

position: static; is used to adjust a design for another medium.

relative

An element can be positioned in relation to its normal position.

To move the element, specify:

  1. position: relative;
  2. a horizontal (x-axis) property (left or right)
  3. a vertical (y-axis) property (top or bottom)

Use directional properties as needed. (If the element only needs to be moved on the x-axis, there is no need to include a y-axis property.)

Without directional properties, the element will note move (but it now has a magical quality which we will see later).

fixed

An element can be positioned in relation to the browser window.

To move the element, specify:

  1. position: fixed;
  2. a horizontal (x-axis) property (left or right)
  3. a vertical (y-axis) property (top or bottom)

Use directional properties as needed.

absolute

An element position as absolute acts like a fixed element, but relative to the nearest positioned ancestor.

If its ancestors are static, the element will be positioned in relation to body.

To absolutely position an element:

  1. the containing element (ancestor) must be position: relative;
  2. the element to be moved is set to position: absolute;
  3. a horizontal (x-axis) property (left or right)
  4. a vertical (y-axis) property (top or bottom)

z-index

Positioned elements can also be controlled in three-dimensional space from front to back.

Higher z-index values will place an element closer to the front.

Tip: increment z-index values by 5 or 10 in case you need to add another element in the future!
