Fonts

Styling Text

Simply changing a typeface can improve the effectiveness of a design.

With web fonts, designers have access to a greater selection of fonts and can craft a meaningful and effective visual message.

Fonts in CSS

There are three ways to specify fonts in CSS:

  • generic styles
  • user fonts
  • web fonts

Generic Styles

Instead of picking a specific typeface, we can choose from one of five generic styles which are determined by browser settings:

  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

Generic Styles: Example

p {
  font-family: serif;
}

User Fonts

The fonts on a users' computer allow us to provide a more tailored design. The typefaces installed on a user's machine may vary.

We can specify primary, secondary and generic choices in our font stack, which offer a fallback if a user does not have a specific font installed.

User Fonts: Example

p {
  font-family: "Andale Mono", "Courier New", monospace;
}

The name of a user font must be enclosed in quotes.

Commas separate each font.

Always specify a generic fallback in case the first choices are unavailable.

Web Fonts

CSS3 introduced the @font-face property, which imports typefaces hosted on our own server or in the cloud.

This development provides designers with thousands of choices.

Adjusting to Web Fonts

Web fonts are relatively new, so the industry is still adjusting to them. Their use is commonplace, but there are caveats:

  • not all browsers support the same file format
  • not all font foundries have licensed fonts for web use
  • fonts hosted in the cloud may load slowly (or not at all)

Where to Find Web Font

Google Fonts provides open source fonts and free hosting!

Font Squirrel offers web fonts for download to be hosted on your site.

Fonts.com, Typography.com and Adobe TypeKit offer pay/subscription services.

(For this course, stick to a free/open-source option.)

I like font X. Can I use it on the web?

Technically, any font can be transformed into a web font.

However, many typefaces do not include licenses which permit Web use.

Before using any font on the web, be sure the license permits you to do so.

(To use a font on the Web without the permission of its foundry would be akin to stealing.)

Google Fonts

Google Fonts offers 900-plus open-source fonts for use.

  1. Click the red circle within a typeface's box to add it to your collection.
  2. Click the Family Selected tab in the bottom right to display options.
  3. Under Customize choose the font weights you wish to use.
  4. Under Embed, select @import.
  5. Copy the line which begins with @import. Do not include the <style> tags. Add this to the very top of your CSS file.
  6. Under Specify in CSS, copy the font-family declaration. Add this to the CSS block in which you want to change a font.

Always specify a fallback in case Google Fonts fails to load.

Google Fonts: Example

@import
  'https://fonts.googleapis.com/css?family=Open+Sans';

body {
  font-family: "Open Sans", sans-serif;
}
