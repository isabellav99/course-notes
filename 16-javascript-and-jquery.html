JavaScript & jQuery

JavaScript

  • the secret sauce of web interactivity
  • introduced in Netscape Navigator 2.0 (1995)
  • commonly used for client-side (browser) scripts

jQuery

  • a library written in JavaScript which simplifies web interaction
  • provides a standard way of working with multiple/older browsers
  • includes a large ecosystem of add-on libraries, plugins

Using jQuery

  • use the 'compressed' version
  • load the libary in HTML
    • include the file on your web server
    • linked to a Content Delivery Network (CDN)
  • include plugins or custom scripts
    • embedded using the <script> tag

Adding jQuery to HTML

If the library is on your server:

<script src="js/jquery-1.11.3.min.js"></script>

If linked from a CDN: 

<script src="https://code.jquery.com/jquery-3.3.1.min.js">
</script>

Note: A link to a page with approved CDNs are included in the example README.

Adding Scripts

jQuery requires a page to be fully loaded before it can work.

Place scripts at the bottom of a page, just inside the closing </body> tag. 

  ...
  <script src="js/jquery-3.3.1.js"></script>
  <script src="js/my-script.js"></script>
</body>

Order Matters

When using plugins, list them in the following order:

  1. jQuery library
  2. plugin libraries
  3. plugin-related settings
  4. your custom scripts
