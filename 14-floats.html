Floats

What is a float?

The float property takes an element out of its normal flow by moving it left or right and wraps text or inline elements around it.

A floated element will be moved all the way to the left or right of its parent.

Float values

left moves the element to the left of its containing block.

right moves the element to the right of its containing block.

none disables any floating. (This is used to adjust a design for another medium.)

Container Collapse

When we float an element, the parent element collapses.

We can fix this by adding the following to the container's CSS block:

overflow: auto;
height: auto;

If the parent element has no specific height set, adding height: auto; is optional.

Fixing white space

Any content next to a floated element has no white space between it and the element.

Good design principles suggest we should fix this.

Add margin to the floated element.

(If the element is floated left, add margin to the right; if floated right, add margin to the left.)

Clearing floats

The clear property stops the effects of a float and re-establishes the normal flow of elements.

Add the it to the element which you want reestablish the normal flow.

left stops a float to the left side.

right stops a float to the right side.

both stops floats to the left and/or right side.
